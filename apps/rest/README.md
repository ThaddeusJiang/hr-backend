# Mojito-Admin Functions

## Setup

```
cp sample.settings.json local.settings.json
```

[local.settings.json](./sample.settings.json)
ask @ThaddeusJiang for the Secrets.

## Install

```
yarn install

yarn start
```

## Test

```
yarn test
```
